# Legal Resources Directory

An Electron desktop application that provides a comprehensive directory of legal resources, helping users find lawyers, legal aid, and support services.

## Features

- **25+ Curated Legal Resources**: Including free legal aid, online services, bar associations, pro bono services, and emergency help
- **Smart Filtering**: Filter by category, cost (free/low-cost/paid), and service type
- **Search Functionality**: Quick search across all resources
- **Modern UI**: Clean, accessible interface with dark/light theme support
- **Secure External Links**: All links open safely in your default browser
- **Responsive Design**: Grid and list view options

## Installation

```bash
# Clone the repository
git clone https://github.com/GuildMasterDev/Lawyer.git
cd Lawyer

# Install dependencies
npm install

# Start the application
npm start
```

## Resource Categories

- **Free Legal Aid**: Government and nonprofit legal assistance programs
- **Online Services**: Digital legal platforms and document services
- **Bar Associations**: State and local bar referral services
- **Pro Bono Services**: Volunteer lawyer programs
- **Legal Research**: Law libraries and research tools
- **Emergency Help**: Crisis hotlines with legal support
